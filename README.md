# Nice Gadgets Store 📱

A modern, responsive e-commerce web application for browsing and shopping for electronics. Built with a focus on performance, clean code, and great user experience.

## 🚀 About the Project
Nice Gadgets Store is a comprehensive product catalog application. It features a modern interface, interactive product navigation, and responsive design, allowing users to browse smartphones, tablets, and accessories seamlessly on any device.

### Core Features:
*   **Product Catalog**: View products with advanced filtering and sorting functionality.
*   **Interactive Slider**: Smoothly animated promotional banner for featured items.
*   **Shopping Cart**: Add, remove, and manage items in your cart.
*   **Favorites**: Save products to a wishlist for later.
*   **Fully Responsive**: Optimized for mobile, tablet, and desktop viewports.

## 🛠 Tech Stack
*   **Framework**: React (Vite)
*   **Language**: TypeScript
*   **Styling**: SCSS / Bulma
*   **Routing**: React Router DOM
*   **Code Quality**: ESLint, Prettier, Stylelint
